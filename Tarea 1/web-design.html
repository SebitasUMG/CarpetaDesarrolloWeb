Descripción del Curso

Este curso introduce los fundamentos del desarrollo web, incluyendo HTML, CSS y JavaScript, así como conceptos de diseño responsivo y accesibilidad web.

Temas Principales

  • HTML5 y estructura semántica
  • CSS3 y diseño responsivo
  • JavaScript básico
  • Frameworks front-end (Bootstrap)
  • Introducción a back-end
  • Despliegue de sitios web
